# Yenuit Solutions — Marketing Website

The official website for **Yenuit Solutions Pvt Ltd** and its flagship product **PayFlow Pro** — a digital payment voucher management system for SMEs.

🌐 **Live:** https://yenuit.com (or your Vercel preview URL)
💳 **Live Demo App:** https://payflow-psms.vercel.app

## Files

- `index.html` — main landing page (home, features, pricing, signup wizard)
- `yenuit-about.html` — "Our Story" page

## Features

- Responsive marketing site with 3-step signup wizard
- Card payment form (Stripe-ready placeholder)
- Bank transfer option
- 30-day trial activation (stored in browser localStorage)
- Admin edit mode (password-protected) for quick content updates
- Yenuit product family section with future product placeholders

## Deployment

Deployed automatically via Vercel. Every push to the `main` branch triggers a new deployment.
